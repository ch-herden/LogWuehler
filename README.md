LogWuehler
==========

### System requirements
+   PHP 5.4 or later
+   Apache HTTP Server with mod_rewrite

### Supported log files
+   Apache Error Log
+   Apache Access Log

### Installation
Download LogWuehler and extract it to the folder you would like to keep it in.
Set the DocumentRoot to the /public folder.
